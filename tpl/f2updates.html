Fiesta 2 V1.40 -> V1.40 (H1.0)
This is our first update! This works with any 1.40 machine.
What this does:
  • - Supports all official future updates.
  • - Unlocks Latin America songs for all regions.
  • - Enables Rush Mode for all songs
  • - Compatible with ANY future patches from Andamiro
The game wont look different inside. It will just unlock the latin america songs (Danza Kuduro and so) for non latin america games. And rush for ANY song.

One Important thing: After the first reboot, remove the pendrive or it will keep updating every time.
This happens because our update is from 1.40 -> 1.40, so the game doesnt know it has been already updated.
It wont harm your machine if it does more than one time, but only removes when the machine is rebooting.
