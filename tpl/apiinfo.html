API Information

How the API works?

POST/GET Request at {SITE_URL} , the response will be at JSON.
You need to make a request with arguments page as api and cmd as the command you want to perform.
The parameters can be both POST and GET requests.

What are the available commands?

See examples for more details
  • savedecode - Decodes the save and send back the data.
  • exptop100 - Returns the Experience TOP 100
  • caloriestop100 - Returns the Calories TOP 100
  • scoretop100 - Returns the Score TOP 100
  • missiontop100 - Returns the Mission TOP 100
  • profile - Returns the saved profile on our servers
  • genball - Generates a Level Ball PNG
  • ucs - Returns a JSON UCS data used by PIU Visual
  • getucsfile - Returns the UCS file
  • mostplayedsongs - Returns the most played songs
  • mostplayedcharts - Returns the most played charts

Error codes

  • 0 - Unknown Error
  • 1 - One or more files are missing
  • 2 - Error uploading file
  • 3 - Invalid file sizes
  • 4 - Invalid CRC
  • 5 - No such profile
